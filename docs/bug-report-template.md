# Bug Report Template

Use this mainly in the `website bugs and testing` thread.

## Fast Template

```text
mode: review only

Bug / issue:

Where it happened:

Route / page:

What I saw:

What I expected:

How to reproduce:
1.
2.
3.

Prompt / input used:

Browser / device:

Console errors:

Network errors:

Screenshot:
```

## Notes
- If the bug is AI-related, include the exact prompt if possible.
- If the bug is visual, include a screenshot.
- If the bug is browser-specific, name the browser and device.
- If there are console or network errors, include them because they speed up triage a lot.
