# Thread Starters

Use these copy-paste starters to keep work in the right thread fast and clean.

## 1. Coding
```text
mode: fix and ship

Task:

Where:

Expected result:

Commit and push after done.
```

## 2. Website Bugs And Testing
```text
mode: review only

Bug / issue:

Where it happened:

What I saw:

What I expected:

How to reproduce:

Prompt / input used:

Browser / device:

Console errors:

Network errors:

Screenshot:
```

## 3. Branding And Design
```text
mode: brainstorm only

Area:

What feels wrong:

What vibe I want:

Reference:
```

## 4. Socials
```text
mode: brainstorm only

Platform:

Post / reel / carousel:

Goal:

Tone:
```

## Quick Router
- `code / commit / push` -> `coding`
- `bugs / QA / server / testing` -> `website bugs and testing`
- `visual direction / branding / design taste` -> `branding and design`
- `captions / reels / launch posts / social plan` -> `socials`
