# OpenAI Media Outputs

Generated OpenAI image and video assets are saved under this folder when using:

- `scripts/openai-media-cli.mjs`
- `scripts/openai-media-mcp.mjs`

Default paths:

- images: `social-assets/openai-media/images/`
- videos: `social-assets/openai-media/videos/`

These generated assets are ignored by git by default.
