# Thread Rules

## 1. Branding And Design
Use this thread for:
- visual direction
- layout ideas
- homepage concepts
- logo/favicon discussions
- design reviews

Do not use this thread for:
- code implementation
- deploy/debug details

## 2. Website Bugs And Testing
Use this thread for:
- manual QA
- server issues
- browser issues
- bug reporting
- Playwright/test observations

Do not use this thread for:
- long feature planning
- social content work

## 3. Coding
Use this thread for:
- feature implementation
- refactors
- code cleanup
- commit
- push

Default expectation in this thread:
- code the task
- commit it
- push it

## 4. Socials
Use this thread for:
- captions
- reels
- launch content
- post order
- creative direction for social media

Do not use this thread for:
- live website bug fixing
- repo refactors

## Shared Rule For All Threads
- If something is for later, save it in `later upgrades/`.
- Keep one thread focused on one kind of work.
- After a big milestone, starting a fresh thread is okay.

## Best Task Format
- what to change
- where it is
- what the final result should feel like
- whether it should be saved for later or implemented now
