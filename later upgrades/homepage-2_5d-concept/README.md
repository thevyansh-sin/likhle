# Homepage 2.5D Concept

This folder stores the premium homepage concept that was approved as a strong future direction for Likhle.

Files:

- `homepage-2_5d-concept-preview.png` — visual reference image
- `homepage-2_5d-concept.html` — concept source layout

Main idea:

- keep Likhle as a tool-first website
- avoid heavy fake 3D
- use a 2.5D feel with layered depth, glow, and cleaner section rhythm
